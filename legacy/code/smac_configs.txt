This file contains the SMAC-optimised configurations (36 hours)
of the methods that use features. Many of these methods contained
further AutoML elements in the form of Auto-sklearn or SMAC. In 
the case of WP-MRP, the use of Auto-sklearn versus using a simple
SVM regressor was also a design choice; in regression methods
directly predicting the target variable, using a simple model as
opposed to Auto-sklearn was found to be infeasible, and thus these
methods do not have separate configurations for Auto-sklearn and
SVM.

The organisation of this file is as follows:

            Parameter2: setting2
